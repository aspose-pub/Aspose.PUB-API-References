## Классы

| Учебный класс | Описание |
| --- | --- |
| [Coord](./coord) | Этот класс предназначен для представления данных, связанных с координатами полей PUB. Каждое поле в PUB содержит координаты, имеющие 2 пары координат: координаты верхнего левого угла (XLeft, YTop) и координаты нижнего правого угла (XRight, YBottom) . Все координаты представлены в специальной метрической системе - English Metric Unit (EMUs). В этот класс были добавлены дополнительные методы для преобразования значений координат из английских метрических единиц в дюймы. |
